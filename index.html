Run API Locally

cd ModernizedApi
dotnet run
# API on http://localhost:5178

Run API with Docker

Build:
cd ModernizedApi
docker build -t modernized-api:latest .
Run:
docker run --rm -p 8080:8080 modernized-api:latest
# API on http://localhost:8080

Serve Minimal Frontend

cd FrontendMinimal
python -m http.server 5500
# Open http://localhost:5500/FrontendMinimal/
